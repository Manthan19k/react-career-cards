# 🚀 React Job Listing UI

A modern and responsive **Job Listing UI** built using React.
This project displays job cards for top companies like Meta, Amazon, Apple, Netflix, etc., with key details such as role, salary, job type, and location.

---

## 📌 Project Overview

This project is designed to practice and demonstrate **React fundamentals** by building a clean and reusable UI for job listings.
Each job is represented as a card component, making the UI modular and scalable.

---

## ✨ Features

* 📄 Clean and modern job card UI
* 🏢 Company branding (logos & names)
* 💼 Job role and level (Junior / Senior)
* ⏱️ Job type (Full-time / Part-time)
* 💰 Salary display
* 📍 Location information
* 🔘 Apply button (UI-based)
* ♻️ Reusable React components
* ⚡ Fast performance using Vite

---

## 🛠️ Tech Stack

* **React.js** – Frontend library
* **Vite** – Fast build tool
* **JavaScript (ES6+)**
* **CSS** – Styling and layout
